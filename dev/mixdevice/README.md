# Mix Device

A library written in *Rust* to communicate with **iOS** devices and **android** devices using native protocols.

> Inspired by **dev/Mobile/DeviceBridge** inside the project [kaleido3d][1].

Support following features:

* [x] device log polling
* [ ] screen shot
* [ ] file upload and download
* [ ] shell command execution
* [ ] application installation
* [ ] socket stream forwarding
* [ ] performance monitor (android perfd & iOS instruments)
* [ ] iOS debug proxy

[1]:https://github.com/PixPh/kaleido3d
